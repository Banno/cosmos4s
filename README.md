## Quick Start

To use cosmos4s in an existing SBT project with Scala 2.12, 2.13 or 3.0, add the following dependencies to your
`build.sbt` depending on your needs:

```scala
libraryDependencies ++= Seq(
  "com.banno" %% "cosmos4s" % "<version>"
)
```
